- #### 项目描述：

  基于vue+springboot框架搭建。通过学习大佬作品搭建的一个个人博客项目，主要功能包括登录系统，阅览、撰写博客，对所有博客通过标签分类、查找、编辑、删除等操作。

- #### 技术栈：

- 前端：Vue + vue-router + Vuex + Element UI + axios

- 后端：Springboot + MyBatis + Redis + MySQL
